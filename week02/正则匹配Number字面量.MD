# 正则匹配Number字面量

```js
/^(\.\d+|(0|[1-9]\d*)\.?\d*?)([eE][-\+]?\d+)?$|^0[bB][01]+$|^0[oO][0-7]+$|^0[xX][0-9a-fA-F]+$/
```
## 整数
```js
/^-?[0-9]+$/g
```
## 浮点数
```js
/^[-+]?[0-9]*\.?[0-9]+$/g
```
## 二进制数
```js
/^[01]+$/
```
## 八进制数
```js
/^[0-7]+\$/
```
## 十六进制数
```js
/(^0x[a-f0-9]{1,2}$)|(^0X[A-F0-9]{1,2}$)|(^[A-F0-9]{1,2}$)|(^[a-f0-9]{1,2}$)/g
```
